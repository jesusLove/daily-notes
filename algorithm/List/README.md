## 4. 删除链表倒数第 N 个结点

### 两次遍历法

思路：第一计算链表长度 sum；第二次遍历找到待删除结点的前驱结点（sum - n)。

### 栈

思路：将链表所有结点压栈，弹出 N 个结点，此时栈顶结点为待删除的前驱结点。

### 双指针

思路：两指针间隔 N 个结点，快指针都末尾时，慢指针指向待删除的前驱结点。
